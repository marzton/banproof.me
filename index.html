// How it works

Four steps from request
to verified signal.

01 — ⬡

Request intercepted

Every inbound request hits the Toll Booth middleware before reaching your API. Bearer token extracted, IP captured.

02 — ◈

PoA validation

Proof of Agency check: token verified against env secrets, IP cross-referenced against the live Tailscale DePIN registry.

03 — ◉

Risk scored

Each request receives a DRS score 0–100. LOW/MEDIUM/HIGH risk levels gate access. High-risk traffic is rejected with reason.

04 — ⬡

Clean delivery

Verified agents pass through with their PoA score injected as a header. Downstream services can trust the payload.
